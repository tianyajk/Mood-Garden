# AI情绪花园（Mood Garden）视觉设计方案 V1.0

> 配套：[需求文档.md](需求文档.md) · [架构设计.md](架构设计.md)
> 设计关键词：治愈 · 梦幻 · 自然 · 高级感
> 参考基因：Journey（情绪叙事 + 玻璃质感）、Apple（克制留白 + 精准动效）、Notion（柔和中性 + 排版秩序）、Forest App（治愈插画 + 成长陪伴）

---

## 设计哲学

> 「每一种情绪都值得被看见。」

界面是情绪的容器，不是信息的仪表盘。设计上做减法——**大留白、弱边框、轻投影、慢动效**——让视觉重心始终落在花园本身。
取 Apple 的克制与精度、Notion 的排版秩序、Journey 的玻璃质感与情绪色温、Forest 的治愈插画与成长叙事，四者融合为「梦幻自然 + 高级感」。

三条铁律：
1. **柔，但不糊** —— 用圆角、低饱和、毛玻璃营造柔和，但层级、对齐、间距必须精准。
2. **静，才显贵** —— 默认安静，动效只在交互与转场时出现，宁慢勿快。
3. **色随情绪走** —— UI 主色是中性的，强调色由当天情绪注入，界面会"呼吸"。

## 二、字体方案

### 2.1 字体家族

| 角色 | 西文 | 中文 | 理由 |
|---|---|---|---|
| 标题 Display | `Fraunces`（衬线，柔润） | 思源宋体 `Source Han Serif` | 衬线带文学治愈感（Journey 气质） |
| 正文 Body | `Inter` | 思源黑体 `Source Han Sans` / 苹方 | 中性清晰（Notion/Apple 气质） |
| 数字 / 日期 | `Inter` 等宽变体 tabular-nums | — | 时间轴对齐 |

> 降级栈：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`。
> 比赛环境若不便引入字体文件，标题可退为系统衬线，正文用 `-apple-system`，仍保持气质。

### 2.2 字号阶梯（1.250 Major Third + 8pt 基准）

| Token | px / line-height | 字重 | 用途 |
|---|---|---|---|
| `display` | 48 / 1.1 | 500 | 首页主标题 Mood Garden |
| `h1` | 32 / 1.2 | 600 | 页面标题 |
| `h2` | 24 / 1.3 | 600 | 区块标题 |
| `h3` | 20 / 1.4 | 600 | 卡片标题 |
| `body-lg` | 18 / 1.6 | 400 | 引导语、陪伴语 |
| `body` | 16 / 1.6 | 400 | 正文 |
| `caption` | 14 / 1.5 | 400 | 辅助、日期 |
| `micro` | 12 / 1.4 | 500 | 标签、计数 |

排版规则：中文正文 `letter-spacing: 0.02em`；标题 `-0.01em`；陪伴语/引导语用斜体衬线增强叙事感。

## 三、首页布局

> 目标：3 秒内传递"治愈 + 梦幻"，一个明确入口。参考 Forest 启动页的留白与 Apple 的居中聚焦。

```
┌──────────────────────────────────────────────┐
│   �────  全屏渐变天空背景 (#7FB89B→#A99BD4)  ────  │
│            缓慢漂浮的低多边形云 / 光点粒子           │
│                                                │
│                    🌱  (Logo)                   │
│                                                │
│              Mood  Garden                       │  ← display 48 衬线
│        记录情绪，种下属于自己的花。                  │  ← body-lg 斜体
│                                                │
│              ╭───────────────╮                  │
│              │   进入花园  →   │   ← 毛玻璃主按钮    │
│              ╰───────────────╯                  │
│                                                │
│         · · ·  （底部极淡版本号/引导）  · · ·        │
└──────────────────────────────────────────────┘
```

规格：
- **垂直居中**，内容宽度 `max-w-[420px]`，整体上浮约 8%（视觉黄金分割）。
- **Logo**：一颗发芽种子插画 / 低多边形小苗，带极慢呼吸缩放（scale 1→1.03，6s 循环）。
- **主按钮**：毛玻璃 `backdrop-blur(16px)` + 半透明白 `rgba(255,255,255,.25)` + 1px 高光描边；hover 时微微发光（box-shadow 扩散 brand/glow）。
- **背景**：Three.js 渐变天 + 5～8 个缓慢漂浮粒子；移动端降级为 CSS 渐变 + SVG 云。
- **入场**：标题 fade-up，副标题延迟 0.2s，按钮延迟 0.4s，云层视差缓入。

## 四、花园页布局（核心视觉页）

> 全屏沉浸式 3D 舞台，UI 以"悬浮玻璃岛"形式漂在场景之上，绝不遮挡花园主体。参考 Journey 的卡片悬浮 + Apple 沉浸式全屏。

```
┌──────────────────────────────────────────────────┐
│ ☀/🌙 昼夜              [今日 · 05.31]      [时间轴]  │ ← 顶部浮条(毛玻璃,半透明)
│                                                    │
│                                                    │
│                  ✿  动态植物 / 粒子                  │
│              （Three.js 全屏 3D 花园）                │
│            🦋        ✨        🌸                    │
│                                                    │
│   ╭────────────────────────╮                       │
│   │  AI 反馈卡 (毛玻璃)        │   ← 左下悬浮          │
│   │  关键词 · 强度 · 反馈      │                      │
│   ╰────────────────────────╯                       │
│                                                    │
│           ❝ 允许自己偶尔迷茫… ❞   ← 底部陪伴语(居中,弱) │
│                          ╭──────────╮               │
│                          │ 今天记录  │ ← 右下主行动    │
│                          ╰──────────╯               │
└──────────────────────────────────────────────────┘
```

规格：
- **层级**：背景天空 → 地形/植物 → 粒子特效 → UI 玻璃层（z 最高，统一 `backdrop-blur` + 低投影）。
- **顶部浮条**：左昼夜指示、中今日日期、右时间轴入口；半透明，向下滚动/无操作 5s 后自动淡隐，鼠标移动唤回（参考 Apple 沉浸式）。
- **AI 反馈卡**：左下角，宽 `320px`，圆角 `20px`，毛玻璃；卡片描边取当天情绪强调色；首次进入 fade-up 弹入。
- **陪伴语**：底部居中，衬线斜体，`body-lg`，透明度 70%，点击可刷新（轻微闪动）。
- **主行动按钮**：右下"今天记录"，胶囊毛玻璃，常驻；已记录则变为"查看今日"。
- **音乐控制**：左下角极简圆形图标，默认静音待用户开启（避免突兀）。
- **响应式**：移动端 UI 玻璃层改为底部抽屉式，3D 场景降低粒子数量与阴影质量。

## 五、动效设计

> 总原则：**慢、柔、有机**。缓动统一用自定义 ease，禁用线性与生硬回弹。情绪类产品的动效是"呼吸"，不是"弹跳"。

### 5.1 缓动与时长 token

| Token | 曲线 | 时长 | 用途 |
|---|---|---|---|
| `ease/soft` | `cubic-bezier(0.4, 0, 0.2, 1)` | 300ms | UI 通用 |
| `ease/gentle` | `cubic-bezier(0.25, 0.1, 0.25, 1)` | 500ms | 卡片 / 转场 |
| `ease/breathe` | `ease-in-out` | 4–6s loop | 呼吸 / 漂浮 |
| `spring/soft` | Framer spring `stiffness 120, damping 18` | — | 选中、弹入 |

### 5.2 关键动效清单

| 场景 | 动效 | 实现 |
|---|---|---|
| 进入花园 | 花朵从地面逐渐绽放（错峰 stagger 80ms） | Three.js scale + Framer 编排 |
| 页面转场 | fade + 轻微上浮 / 视差，不切硬边 | `PageTransition`（Framer AnimatePresence） |
| 鼠标移动 | 粒子柔性跟随（带阻尼延迟） | `ParticleCursor` |
| 点击花朵 | 该花轻微放大 + 浮出当天心情卡 | raycast + spring/soft |
| 情绪卡选中 | 卡片上浮 4px + 强调色描边渐显 + 微光 | spring/soft |
| 提交记录 | 按钮收缩 → 种子落地 → 场景生长一帧 | 链式编排 |
| 夜晚模式 | 萤火虫随机游走 + 整体降亮升紫 | shader / 粒子系统 |
| 雨天模式 | 雨滴下落 + 地面涟漪 + 雾气渐入 | 粒子 + 后期 |
| 陪伴语刷新 | 旧句淡出上移、新句淡入 | Framer |

### 5.3 性能与无障碍

- 帧率目标 60fps；移动端粒子数 / 阴影自动降级。
- 尊重系统 `prefers-reduced-motion`：关闭漂浮/视差，仅保留 opacity 过渡。
- 所有 loop 动效可被音乐/性能开关统一暂停。

## 六、设计语言（Design Language）

可直接映射到 `config/theme.ts` 与 Tailwind config，形成统一 design token 体系。

### 6.1 形状 · 圆角
| Token | 值 | 用途 |
|---|---|---|
| `radius/sm` | 12px | 输入框、标签 |
| `radius/md` | 20px | 卡片、按钮 |
| `radius/lg` | 28px | 浮层、玻璃岛 |
| `radius/full` | 9999px | 胶囊、头像、图标钮 |

> 全局偏大圆角营造柔和；绝不用直角，弱化锐利感。

### 6.2 间距（8pt 栅格）
`4 · 8 · 12 · 16 · 24 · 32 · 48 · 64`；卡片内边距默认 `24`，区块间距 `48`，页面边距移动端 `20` / 桌面 `40+`。

### 6.3 投影（轻 · 暖 · 弥散）
| Token | 值 |
|---|---|
| `shadow/sm` | `0 2px 8px rgba(75,138,110,.06)` |
| `shadow/md` | `0 8px 24px rgba(75,138,110,.10)` |
| `shadow/glow` | `0 0 40px rgba(246,231,193,.45)`（情绪微光） |

> 投影带极淡绿色温，而非冷灰，强化"自然"基调。

### 6.4 毛玻璃（高级感核心材质）
```
background: rgba(255,255,255,0.18);
backdrop-filter: blur(16px) saturate(120%);
border: 1px solid rgba(255,255,255,0.35);
box-shadow: shadow/md;
```
深色场景（夜晚）切换为深色玻璃：`rgba(30,34,55,0.35)` + 浅色描边。

### 6.5 图标 · 插画
- 图标：线性、圆头端点、`1.5px` 描边，统一 24×24（Apple SF 气质）。
- 插画：低多边形 / 治愈系扁平，弱描边、柔渐变、Ghibli 色温。
- 情绪 emoji 仅用于选择卡，正式场景一律用定制插画，保持高级感。

### 6.6 组件状态规范（可达性）
- 文字对比度 ≥ 4.5:1（暖黑 ink/900 over 米白达标）。
- 交互元素 hover/active/focus/selected/disabled 五态齐全；focus 用 `2px` 情绪强调色外环。
- 触控目标 ≥ 44×44px。

### 6.7 语气 · 文案
克制、温柔、第二人称。例：「今天想记录些什么？」「这一刻的情绪，已经被看见了。」避免感叹号轰炸与营销腔。

## 七、Design Token 落地映射（给开发）

| 设计概念 | 落点 | 说明 |
|---|---|---|
| 中性色 / 品牌色 / 语义色 | `tailwind.config.ts` theme.colors | 全量 token 化 |
| 情绪强调色 | `config/emotions.ts` 的 `color` 字段 | UI 柔化版与场景原色分离 |
| 圆角 / 间距 / 投影 | `tailwind.config.ts` extend | 与本文 6.1–6.3 一致 |
| 缓动 / 时长 | `config/theme.ts` motion token | Framer 与 CSS 共用 |
| 毛玻璃 | 抽成 `ui/Card` 的 `glass` 变体 | 复用，避免重复样式 |

### 一句话总结
**米白底 + 自然绿到蓝紫的梦幻渐变 + 毛玻璃玻璃岛 + 柔润衬线标题 + 慢呼吸动效**——安静、高级、会随情绪呼吸的治愈花园。
