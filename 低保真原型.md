## 六、导航与流转总结

| 入口 | 去向 | 触发 |
|---|---|---|
| 首页「进入花园」 | 花园页（或空花园引导记录） | 主按钮 |
| 花园页「今天记录」 | 记录页 | 右下常驻按钮 |
| 记录页「种下今天」 | 提交→AI 分析→返回花园(生长动画) | 主按钮 |
| 花园页「时间轴」 | 时间轴页 | 顶部浮条 |
| 时间轴条目 / 详情「在花园中查看」 | 花园页定位该花 | 点击 |
| 任意子页「←」 | 上一级 | 返回 |

> 所有浮层(▒)统一毛玻璃 + 大圆角；带 ┄ 的元素无操作时自动淡隐，鼠标移动唤回（仅花园页）。
